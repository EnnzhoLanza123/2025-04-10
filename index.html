Série de Posters Modernos

Esta série de posters foi desenvolvida para eventos culturais na minha escola, incluindo feiras de ciências e festivais literários. Cada poster foi cuidadosamente planejado para comunicar claramente o evento enquanto mantém um apelo visual forte.

O processo criativo incluiu pesquisa de referências, esboços manuais e desenvolvimento digital no Photoshop. Foquei especialmente em criar hierarquia visual clara, garantindo que as informações mais importantes fossem imediatamente perceptíveis.

Para a tipografia, combinei fontes serifadas modernas com sans-serif clean, criando contrastes interessantes. As paletas de cores foram selecionadas para refletir o tema de cada evento - cores mais sóbrias para eventos acadêmicos e paletas vibrantes para atividades culturais.

Este projeto foi importante para desenvolver minha compreensão de composição visual e comunicação através do design. Apesar de ser um trabalho escolar, recebi feedback positivo de professores e colegas, o que me motivou a continuar explorando o design gráfico.

Design de Interface para App Escolar

Este projeto de design de interface foi desenvolvido como parte de uma atividade interdisciplinar que combinava tecnologia e design. O objetivo era criar um aplicativo que ajudasse os alunos a organizarem suas tarefas escolares e atividades extracurriculares.

Comecei com pesquisa de usuário, entrevistando colegas para entender suas necessidades e frustrações com os aplicativos existentes. Em seguida, criei personas e fluxos de usuário para guiar o processo de design.

Os wireframes foram feitos inicialmente no papel, depois refinados digitalmente. Para os mockups finais, utilizei uma abordagem minimalista com foco na usabilidade. A navegação foi projetada para ser intuitiva, com ícones claros e consistência visual em todas as telas.

A paleta de cores foi escolhida para ser calma e não distrativa, com azuis suaves que transmitem profissionalismo e confiança. A tipografia foi selecionada para máxima legibilidade em diferentes tamanhos de tela.

Identidade Visual para Banda Amadora

A identidade visual foi criada para uma banda de rock alternativo formada por amigos da escola. O desafio era desenvolver uma marca que refletisse o estilo musical da banda enquanto mantinha versatilidade para aplicação em diferentes materiais.

O processo começou com discussões com os membros da banda para entender sua visão e inspirações musicais. Criei um moodboard com referências visuais antes de partir para os esboços iniciais do logotipo.

O logotipo final combina elementos tipográficos customizados com um símbolo abstrato que remete a ondas sonoras. A identidade visual inclui uma paleta de cores limitada (preto, branco e um vermelho vibrante) para garantir impacto visual forte e reprodução consistente em diferentes mídias.

Além do logotipo, desenvolvi aplicações para capas de singles, perfil nas redes sociais e adesivos. Este projeto foi especialmente valioso para aprender sobre branding e como criar sistemas visuais coesos.
